# Keyboard Check

[中文版本](README_CN.md)

A Rust TUI application that visualizes keyboard input in the terminal.

## Features

- Real-time keyboard visualization
- QWERTY keyboard layout display
- Key press highlighting (green background)
- Support for special keys
- Clean terminal UI

## Installation

```bash
git clone https://github.com/yourusername/keyboard-check.git
cd keyboard-check
cargo build --release
```

## Usage

```bash
cargo run
```

Press any key to see it highlighted on the keyboard layout.
Press ESC or Ctrl+C to exit.

## Development

```bash
# Run tests
cargo test

# Run linter
cargo clippy

# Format code
cargo fmt
```
